About the SurveyBox Module - version 1.2
The SurveyBox DNN® module offers easy integration of the Survey™ Project's front end features into the DotNetNuke® CMS to present surveys, webforms and questionnaires to DNN users, respondents and voters. Creation and administration of surveys is done through the Survey™ Project Webapplication. Users/ respondents can answer and submit surveys throught the SurveyBox DNN® module placed on the pages of the DNN installation.
New Features & Fixes
Improved and more user friendly version of the SurveyBox DNN Module: - Option to select Survey to present from dropdownlist on Module Settings Tab; - Complete installation through DNN® Installer Wizard and DNN® interface; * no need to copy files manually into the DNN® root; * all files included in DesktopModules/SurveyBox directory; * web.config entries added through installer (*);
(*) Note: Nsurveysettings and SP databaseconnection can be set through the DNN Host/Configuration Manager
